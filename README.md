# 一个简单的rbac权限管理系统
使用.net8,sqlsugar,autofac来实现简单的基于角色的权限管理系统
## 项目启动流程
1. 下载项目文件到本地
2. 下载.net8 sdk
3. 使用vs打开项目，并将rbac作为启动项
4. 点击运行
## 配置跨域流程
1. 打开appsetting.json文件
2. 在AllowedHost中增加对应的前端地址
